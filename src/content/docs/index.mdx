## What is Bifrost?

Bifrost is an open-source workflow automation platform designed for MSPs and Integration Services. It provides a powerful way to automate business processes, manage integrations, and create dynamic user interfaces—all without complex infrastructure.

**Key features:**
- Workflow automation with Python
- Dynamic form generation with data providers
- Built-in OAuth and secrets management
- Multi-tenant support with organization isolation
- Enterprise-grade security and audit logging
- Seamless Azure integration
